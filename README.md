
# Installation & Setup
#### 1.Clone the Repository
    git clone https://github.com/LCHIN7X/phishing_URL.git
#### 2.Install Dependencies
    pip install -r requirements.txt
#### 3.Run the application
    python app.py

    http://127.0.0.1:5000/
